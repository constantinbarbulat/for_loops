//=============Loop_for===========

Questo blocco di codice utilizza un ciclo "for" per stampare i numeri da 0 a 5 sulla console.

Ecco come funziona:

Il ciclo inizia inizializzando una variabile chiamata "i" con il valore 0.
Quindi verifica se "i" è minore o uguale a 5. Se questa condizione è vera, esegue il codice all'interno 
del ciclo (in questo caso, stampa il valore corrente di "i" sulla console).
Dopo aver eseguito il codice all'interno del ciclo, incrementa il valore di "i" di 1 utilizzando
 l'operatore "++".
Il ciclo continua questo processo fino a quando "i" non è più minore o uguale a 5.

//=============Loop_for x2===========

Questo blocco di codice utilizza due cicli "for" per stampare i numeri da 0 a 3 e i numeri da -1 a -3 sulla
 console.

Ecco come funziona:

Il primo ciclo "for" inizia inizializzando una variabile chiamata "i" con il valore 0.
Quindi verifica se "i" è minore o uguale a 3. Se questa condizione è vera, esegue il codice all'interno 
del primo ciclo "for" (in questo caso, stampa il valore corrente di "i" sulla console).
Dopo aver eseguito il codice all'interno del primo ciclo "for", il secondo ciclo "for" inizia inizializzando
 una variabile chiamata "j" con il valore -1.

Quindi verifica se "j" è maggiore o uguale a -3. Se questa condizione è vera, esegue il codice all'interno 
del secondo ciclo "for" (in questo caso, stampa il valore corrente di "j" sulla console).
Dopo aver eseguito il codice all'interno del secondo ciclo "for", il valore di "j" viene decrementato 
di 1 utilizzando l'operatore "--".

Il ciclo continua questo processo fino a quando "j" non è più maggiore o uguale a -3.
Dopo aver completato il secondo ciclo "for", il primo ciclo "for" ritorna e il valore di "i" viene
 incrementato di 1 utilizzando l'operatore "++".
Il ciclo continua questo processo fino a quando "i" non è più minore o uguale a 3.

 //=======Asterix piramid===========
Acest bloc de cod utilizează două cicluri "for" pentru a crea un șir de caractere format din caracterele "*". 
Șirul astfel format este afișat pe consolă sub forma unui triunghi cu baza de lungimea "length"
 (în acest exemplu, "length" este setat la 7).

Iată cum funcționează:

Blocul de cod începe prin definirea unei variabile "result" ca un șir gol.
Apoi este definită o variabilă "length" cu o valoare de 7.

În primul ciclu "for", variabila "i" este inițializată cu 0 și verificată dacă este mai mică decât "length".
 Dacă această condiție este adevărată, se execută blocul de cod în interiorul primului ciclu "for",
  care constă într-un al doilea ciclu "for".

În al doilea ciclu "for", variabila "j" este inițializată cu 0 și verificată dacă este mai mică decât "i". 
Dacă această condiție este adevărată, caracterul "*" este adăugat la șirul "result".
După ce al doilea ciclu "for" este terminat, caracterul "\n" (nouă linie) este adăugat la șirul "result". 
Aceasta creează o nouă linie pe consolă.

Primul ciclu "for" continuă până când valoarea lui "i" nu mai este mai mică decât "length". 
În fiecare iterație a primului ciclu "for", se creează o nouă linie formată din caracterele "*" de lungimea "i".
După ce blocul de cod este completat, șirul "result" este afișat pe consolă, 
creând un triunghi format din caracterele "*".
